Profanity Checker
====
*This is a simple python module to detect curse word from a file*

---

## How To Use:

*Replace **quotes.txt** with your own file*

-  If used as a module
   ```python
   from profanity import read_text
   read_text("quotes.txt")
   ```
-  If used as main 
   ```python
   read_text("quotes.txt")
   ```
